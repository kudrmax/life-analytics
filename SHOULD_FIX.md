## Тривиальные auto-sources в корреляциях

`checkpoint_max`/`checkpoint_min` генерируются для метрик с 1 интервалом/чекпоинтом. В этом случае max = min = единственное значение = агрегат. Эти auto-sources порождают тысячи бесполезных пар (2656 пар из 3 метрик).

**Метрики-примеры:** Кофе (кол-во шотов), Зарядка, Время начала работы — у каждой 1 интервал.

**Фикс:** не генерировать `checkpoint_max`/`checkpoint_min` для метрик с `len(bindings) < 2`.
